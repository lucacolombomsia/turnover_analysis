Data ingestion and model fitting¶

makedb.make_tables()¶

Reads the training data from csv. Randomly splits the data in one training set and two test sets. Writes each dataset to a different table in the database.

model.fit_model(data)¶

Fits a logistic regression on the training data that has been read from the database.

Parameters:data (tuple) – Tuple with X matrix and Y vector.
Returns:A trained logistic regression model
model.pickle_model(model)¶

Takes a trained model and writes it into a pickle file.

Parameters:model – A trained model.
__init__.preprocess_for_sklearn(data)¶

Preprocesses the data for all the predictors into a format that is compatible with sklearn, by converting all categorical variables into dummies. The preprocessed data can be used both to train the model and to make predictions on the test set.

Parameters:data (dataframe) – The dataframe with the data to be preprocessed.
Returns:A dataframe with the processed data.
Return type:dataframe
__init__.read_data(table_name)¶

Reads the data from a specified table in the database. It can be used to query both the train and the test set. When asked to read the table that contains the training data, it will return a tuple with the data from all the predictors (the X matrix) and the data for the response (the Y vector). When asked to read a table that contains the test data, it will return a pandas dataframe with the data from all the predictors (the X matrix).

Parameters:table_name (str) – Name of table to be queried
